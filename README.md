```Homeworking: Site Scraping```

## What it does:
This website collects article data from other websites and organizes them for you.

Features:
- Scrape other website for articles
- Save favorite articles
- Remove favorite articles
- Add comments for your favorite articles

## How
### Scraping:
- Access /scrape if you need to pull more data in!
- The website should automatically load all articles and store them in the database upon load.
